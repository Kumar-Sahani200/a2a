## Setup

### 1. Git clone it

```bash
git clone https://github.com/Kumar-Sahani200/a2a.git
```

### 2. Install dependencies

```bash
cd a2a
uv sync
```

### 3. Run the server

```bash
uv run __main__.py
```

### 4. Run the client(in another terminal)

```bash
uv run test_client.py
```
